# TITLE: Differential Geometry
# TYPE: knowledge
# SPACES: geometry, calculus, semantic

## DATA

# Eğriler
curve_parameterization r(t) = (x(t), y(t), z(t))
arc_length s = ∫|r'(t)| dt
curvature κ = |r'(t) × r''(t)|/|r'(t)|³
torsion τ measures deviation from planar curve
frenet_frame (T, N, B) orthonormal basis along curve

# Yüzeyler
surface_parameterization r(u,v) = (x(u,v), y(u,v), z(u,v))
first_fundamental_form I = E du² + 2F du dv + G dv²
second_fundamental_form II = L du² + 2M du dv + N dv²
gaussian_curvature K = (LN - M²)/(EG - F²)
mean_curvature H = (EN - 2FM + GL)/(2(EG - F²))

# Manifoldlar
manifold locally Euclidean space
tangent_space vector space at point p on manifold
riemannian_metric inner product on tangent spaces
geodesic shortest path on manifold
